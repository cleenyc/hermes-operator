# Hermes Operator

Hermes Operator is a portable autonomy and orchestration layer for Hermes Agent. It is
designed to turn Hermes into a proactive personal operator that can maintain durable work
state, coordinate parallel execution, and use live reasoning passes to advance clear-scope
work without losing human control over consequential actions.

This initial repository state documents the project scope before implementation history is
introduced.
